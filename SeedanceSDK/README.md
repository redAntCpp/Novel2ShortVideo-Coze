# Seedance 2.0 视频生成 SDK 快速入门

欢迎使用方舟平台 Seedance 2.0 视频大模型！本教程专为初次使用 API 的开发者设计，旨在帮您免去环境搭建和依赖安装的烦恼，让您能将精力完全集中在视频生成的业务逻辑上。通过我们提供的自动化脚本，您可以**一键配置 Python 开发环境**、创建纯净的虚拟环境并自动安装方舟 SDK (`volcenginesdkarkruntime`)。此外，包内还包含了一份开箱即用的 Seedance 2.0 示例代码，只需简单修改参数，您就能立即开启视频创作之旅。
