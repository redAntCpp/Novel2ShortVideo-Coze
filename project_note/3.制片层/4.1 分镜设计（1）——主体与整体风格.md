# 整体风格与主体

## global_style（整体风格）

这是整部片子的“总导演规则”，它的作用是让所有镜头风格统一。否则会出现：

- 第一镜像王家卫
- 第二镜像漫威
- 第三镜像抖音广告

Json结构如下
```json
"global_style": {
  "visual_style": "日式悬疑电影",
  "color_palette": ["冷蓝", "灰白", "霓虹红"],
  "lighting": "冷色便利店日光灯",
  "camera_style": "handheld",
  "aspect_ratio": "16:9",
  "resolution": "1080p",
  "film_look": "35mm film grain"
}
```

### visual_style

整体的视觉风格：

| 风格（示例） | 影响           |
| ------------ | -------------- |
| 日式悬疑     | 克制、冷清     |
| 港风警匪     | 霓虹、快速推拉 |
| 王家卫       | 慢镜、情绪化   |
| 诺兰         | 压迫感、低饱和 |
| 好莱坞商业片 | 强对比、高动态 |

### color_palette

整体色调：

| 色调   | 情绪                 |
| ------ | -------------------- |
| 冷蓝   | 孤独、悬疑           |
| 橙黄   | 温暖                 |
| 墨绿   | 压抑                 |
| 霓虹红 | 危险、都市、赛博朋克 |

### lighting

光源逻辑

举例：

| 灯光       | 电影感   |
| ---------- | -------- |
| 冷白日光灯 | 现实压抑 |
| 烛光       | 古典     |
| 霓虹灯     | 赛博朋克 |
| 侧逆光     | 情绪化   |
| 顶光       | 压迫     |

### camera_style

默认镜头风格。示例：

| 风格            | 感觉       |
| --------------- | ---------- |
| handheld        | 真实感     |
| tripod          | 稳定纪录片 |
| cinematic dolly | 电影感     |
| drone           | 史诗感     |
| shaky cam       | 紧张感     |

### aspect_ratio 

默认构图规则，示例:

| 比例 | 感觉     |
| ---- | -------- |
| 16:9 | 标准电影 |
| 9:16 | 短视频   |
| 21:9 | 史诗电影 |
| 4:3  | 怀旧压迫 |

### film_look

材质感,示例

| 风格            | 质感     |
| --------------- | -------- |
| film grain      | 胶片     |
| clean digital   | 数码广告 |
| VHS             | 复古     |
| cinematic bloom | 梦幻     |

## characters(角色)

解决角色的一致性问题。

```json
{
	"char_info": {
		"char_id": "",
		"char_name": "",
		"phase_id": "",
		"state_id": ""
	},
	"reference_image": "assets/linxia.png",
	"traits": [
		"黑长发",
		"白衬衫",
		"疲惫感",
		"细框眼镜"
	]
}
```

这里可以引用角色分析层的结果。将角色的参考图引用出来。
