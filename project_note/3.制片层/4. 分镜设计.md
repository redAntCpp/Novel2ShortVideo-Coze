# 分镜设计（1）——总览

## 一、传统分镜的问题

很多人的结构是：

```
{
  "shot": 1,
  "prompt": "一个男人在街头奔跑"
}
```

模型不知道：

- 谁
- 长什么样
- 从哪里开始
- 怎么运动
- 镜头怎么拍
- 节奏如何变化
- 哪部分来自参考图
- 哪部分来自参考视频
- 哪部分必须保持一致

所以生成非常随机。

## 二、分层结构

应该拆成的 8 个层：

1. 主体层（Character Layer）：解决谁是谁
2. 场景层（Spatial Layer）：解决画面空间
3. 时间层（Temporal Layer)：镜头1 → 镜头2 → 镜头3, 按时间顺序组织内容。
4. 运镜层（Camera Layer)：电影感本质很多来自：镜头、焦段、运动，而不是形容词。
5. 动作层（Performance Layer）：动作必须可表演。
6. 音频层（Audio Layer）：对于原生音视频联合生成，所以音频必须结构化。
7.  约束层（Constraint Layer）：非必要
8.  生成策略层（Generation Strategy）：告诉AI整体的生成策略

整体流程：

```tex
剧本
↓
剧情理解Agent
↓
导演Agent
↓
Storyboard JSON
↓
Prompt Compiler
↓
Seedance Prompt
↓
视频生成
```

传统的制作流程：

| 职位 | 管什么     |
| ---- | ---------- |
| 导演 | 表演、镜头 |
| 摄影 | 运镜、焦段 |
| 美术 | 场景风格   |
| 灯光 | 光影氛围   |
| 录音 | 声音       |
| 剪辑 | 节奏       |
| 场记 | 连续性     |
