- 情绪变化
- 状态变化
- 行为变化

- 表情变化
- 动作变化
- 情绪轻微变化

1. 必须强调：
- 人数较多
- 面容普通

- 群体行为（如围观、交谈）

五、情绪基调（稳定层）

- 灵魂变化（人 → 鬼 / 被夺舍长期存在

年龄
阶段名：xxx时期
外观
性格
状态

上台表演 / 扮演角色
易容 / 伪装
工作状态（上班 / 执行任务）
战斗状态
短期附身 / 异常状态
